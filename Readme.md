# Fundament

My own personal Swift "Prelude" library, complete with all my favorite niceties.

## Features

- `Either` type
- `reduce()` on `Optional`
- Functional Optics
    - `Lens`
    - `Prism`
    - `Traversal`
    - `Isomorph`
    - compositon with all optics types
    - use `KeyPath`s as lenses
